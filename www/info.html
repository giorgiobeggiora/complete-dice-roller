Documentation

This app uses an extended version of the classic dice notation.
The basic concept is that "+" keeps and "-" drops.

Classic notation actions
  • 1d100 = roll one 100-sided dice. You can replace 100 with any number.
  • 4d6+H3 = roll four 6-sided dices then keep the highest three. The number after H is optional (default: 1).
  • 4d6-L1 = roll four 6-sided dices then drop the lowest. The number after L is optional (default: 1).
  • 4k6 = 4d10+H6 = roll four 10-sided dices then keep the highest six.
  • 4dF = 4d3-8 = roll four Fudge dices.
Extended notation actions
  • 4k6-R1+R10 = 4d10+H6 then drop and reroll the results equalt to 1, then keep and reroll the results equalt to 10 (exploding dices). The number after R is optional (drop default: 1, keep default: 10)
  • 10d10+S8 = 10d10 then count the results equalt to or greater than 8 (successes). The number after S is optional (default: 8).
  • 10d10-F8 = 10d10 then count the results lower than 8 (failures). The number after F is optional (default: 8).
Super example

10d10-R10-R9-R8+H10+S+1=1 ;)
